Tools included here only need to be (+ Added) in the tool section in OpenWebUI's workspace area, then enabled under Admin Settings>Settings>Model settings

**wayback_archive.py** - When a site is blocked this tool will allow your model to find the most recent backup of the site at archive.org
Instructions for the model are contained in the tool, if it is tool aware you shouldnt need prompting for this but it may help depending on what you're running.

**get_sports.py** - Fetches live scores, upcoming games, and final results with times/dates for sporting events via ESPN free web API
